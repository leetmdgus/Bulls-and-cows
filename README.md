# Bulls-and-cows

+ 최근 수정일 : 2020:10:30

+ 숫자야구란?

  + A가 3자리 수를 생각한다. 각 자리는 달라야 하며, 상대방이 이야기한 수와 위치가 같을 경우 스트라이크, 같지만 위치가 다르면 볼, 모든 수가 맞으면 홈런이다.

    A : 1 2 3

    B:  5 6 7

    A : 0b 0s!

    B:  3 2 1

    A:  1s 2b!

    B: 1 2 3

    A: 3s 0b! 홈런!

  + 사용자의 숫자를 입력받아 프로그램이 맞추고, 사용자도 프로그램의 숫자를 맞추는 프로그램을 작성해보자.
